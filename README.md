# Installation

### To install the package, open R and type:


``` r
install.packages("devtools")
```

### Then, you can install ArgentinaSSP126:


``` r
devtools::install_github("chihlinwei/ArgentinaSSP126", dependencies = TRUE)
library(ArgentinaSSP126)
```
